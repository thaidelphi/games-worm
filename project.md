# Worm Game - System Configuration (`sys_config.js`)

ไฟล์ `server/sys_config.js` เป็นจุดศูนย์รวมในการปรับแต่งค่าสมดุล (Balance) และระบบต่างๆ ภายในเกม โดยมีการตั้งค่าที่สามารถปรับเปลี่ยนได้ดังนี้:

## 🌍 World Settings (ตั้งค่าโลกของเกม)
- `WORLD_WIDTH` (5000): ความกว้างของแผนที่
- `WORLD_HEIGHT` (5000): ความสูงของแผนที่
- `TICK_RATE` (20): อัตราการอัปเดตเซิร์ฟเวอร์ต่อวินาที (Ticks Per Second)

## 🐍 Snake Settings (ตั้งค่างู)
- `BASE_SPEED` (6.0): ความเร็วปกติของงูตอนเลื้อย
- `BOOST_SPEED` (12.0): ความเร็วตอนงูกดปุ่มเร่ง (พุ่ง)
- `INITIAL_LENGTH` (15): ความยาวเริ่มต้นของงูตอนเกิด
- `SEGMENT_DISTANCE` (12): ระยะห่างระหว่างข้อต่อแต่ละชิ้น
- `BASE_RADIUS` (12): ความอ้วนเริ่มต้นของหัวและตัวงู
- `MAX_RADIUS` (35): ความอ้วนสูงสุดที่งูสามารถเติบโตได้

## 🍎 Food Settings (ตั้งค่าอาหาร)
- `FOOD_COUNT_TARGET` (600): จำนวนอาหารปกติที่ระบบจะพยายามรักษาไว้บนแผนที่เสมอ
- `FOOD_RADIUS_MIN` (4): ขนาดความกว้างเล็กสุดของเม็ดอาหาร
- `FOOD_RADIUS_MAX` (9): ขนาดความกว้างใหญ่สุดของเม็ดอาหารปกติ
- `CORPSE_FOOD_LIFESPAN_SEC` (25): อายุของซากอาหารที่ดรอปจากงูที่ตาย (วินาที) ก่อนจะสลายไป

## ⭐ Special Item Settings (ตั้งค่าไอเทมพิเศษ)
- `ENABLE_SPECIAL_ITEMS` (true/false): สวิตช์สำหรับเปิด/ปิด การดรอปไอเทมพิเศษทั้งหมดในเกม
- `SPECIAL_ITEM_DROP_CHANCE` (0.05): โอกาส 5% ที่เม็ดอาหารธรรมดา หรือซากอาหารที่ดรอป จะกลายเป็นไอเทมพิเศษ
- `ITEM_DURATION_X2` (30): ระยะเวลาบัฟคะแนนคูณสอง (วินาที)
- `ITEM_DURATION_X5` (15): ระยะเวลาบัฟคะแนนคูณห้า (วินาที)
- `ITEM_DURATION_X10` (10): ระยะเวลาบัฟคะแนนคูณสิบ (วินาที)

## 🤖 Bot Settings (ตั้งค่าบอท)
- `BOT_COUNT_TARGET` (8): จำนวนบอทเป้าหมายที่จะให้ออนไลน์อยู่ในเซิร์ฟเวอร์เสมอ
- `BOT_NAMES`: รายการชื่อสุ่มสำหรับตั้งชื่อบอท (Array)

## 💻 Client (UI) Settings (ตั้งค่าฝั่งผู้เล่น)
- `BOOST_DRAIN` (0.008): อัตราการลดลงของหลอดพลังงานวิ่งเร็ว (ต่อ Frame)
- `BOOST_REGEN` (0.003): อัตราการฟื้นฟูของหลอดพลังงานวิ่งเร็ว
- `MIN_ZOOM` (0.5): ระยะซูมออกที่ไกลที่สุดที่อนุญาตให้ผู้เล่นเลื่อนลูกกลิ้งเมาส์
- `MAX_ZOOM` (2.0): ระยะซูมเข้าที่ใกล้ที่สุดที่อนุญาตให้ผู้เล่นเลื่อนลูกกลิ้งเมาส์
- `ZOOM_STEP` (0.1): อัตราการปรับซูมต่อการคลิกเมาส์กงล้อ 1 ครั้ง
